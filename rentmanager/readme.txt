commande pour lance le site : mvn tomcat7:run

-home : nombre de client, reservation et vehicule
- 3 menus : utilisateurs, voitures et reservations
-Pour chaque menu, liste des mebres  + 4 actions :
   - ajouter, pour ajouter un membre
   - details, pour détailler les informations d'un membre
   - update, pour modifier les  informations d'un membre
   - delete, pour supprimer un membre
Pour creer un utilisateur, il faut respecter les règles suivantes :
   - avoir pus de 18 ans
   -adresse non existente
